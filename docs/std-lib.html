Standard Library

Built-in modules for common tasks

Modules

  • std::io - Input/Output operations
  • std::http - HTTP client
  • std::json - JSON parsing and serialization
  • std::reflect - Runtime reflection
  • std::sys - System operations

std::io

Input/Output operations for console output and file handling.

println(message: str)

Print a message followed by a newline.

Example 
import std::io

println("Hello, World!")
// Output: Hello, World!
//         (with newline)

print(message: str)

Print a message without adding a newline.

Example 
import std::io

print("Loading...")
print("done!")
// Output: Loading...done!

std::http

HTTP client for making web requests with async support.

HttpClient

A class for making HTTP requests with configurable options.

HttpClient.new(): HttpClient

Create a new HTTP client instance.

client.set_timeout(int ms)

Set the request timeout in milliseconds.

client.set_verify_ssl(bool verify)

Enable or disable SSL certificate verification.

client.add_header(str name, str value)

Add a header to all requests.

Example 
import std::http

// Create HTTP client
let client = HttpClient.new()
client.set_timeout(5000)
client.set_verify_ssl(true)

// Add default headers
client.add_header("User-Agent", "Bengal/1.0")
client.add_header("Accept", "application/json")

HttpMethod (enum)

HTTP method enumeration.

Value Description
HttpMethod.GET HTTP GET request
HttpMethod.POST HTTP POST request
HttpMethod.PUT HTTP PUT request
HttpMethod.DELETE HTTP DELETE request

RedirectPolicy (enum)

Redirect handling policy enumeration.

Value Description
RedirectPolicy.Follow Automatically follow redirects
RedirectPolicy.None Do not follow redirects

async fn get(str url): str

Make an asynchronous HTTP GET request and return the response body.

Example 
import std::http

async fn fetch_example(): str {
  let response = await http.get("https://api.example.com/data")
  return response
}

std::json

JSON parsing and serialization utilities.

stringify(any value): str

Convert a value (object, array, or primitive) to a JSON string.

Example 
import std::json

class User {
  name: str = "Alice"
  age: int = 30
}

let u = User()
let jsonStr = std::json::stringify(u)
println("JSON: ${jsonStr}")
// Output: {"name":"Alice","age":30}

parse(str json): any

Parse a JSON string and return the corresponding value.

Example 
import std::json

let jsonStr = '{"name":"Bob","age":25}'
let parsed = std::json::parse(jsonStr)

println("Name: ${parsed.name}")
println("Age: ${parsed.age}")
Tip: The parse function returns an object that can be accessed using dot notation for properties.

std::reflect

Runtime reflection and type inspection utilities.

type_of(any value): str

Get the type name of a value as a string.

Example 
import std::reflect

class User {
  name: str = "Alice"
}

let u = User()
let type = std::reflect::type_of(u)
println("Type: ${type}")
// Output: Type: User

class_name(any value): str

Get the class name of an object as a string.

Example 
import std::reflect

class User {
  name: str = "Alice"
}

let u = User()
let className = std::reflect::class_name(u)
println("Class: ${className}")
// Output: Class: User

fields(any obj): any

Get all fields of an object as a map-like object.

Example 
import std::reflect

class User {
  name: str = "Alice"
  age: int = 30
}

let u = User()
let fields = std::reflect::fields(u)

println("Name: ${fields.name}")
println("Age: ${fields.age}")
Note: The fields function returns an object that can be accessed using dot notation to get field values.

std::sys

System-level operations and environment access.

env(str name): str

Get the value of an environment variable.

Example 
import std::sys

// Get PATH environment variable
println("PATH: " + std::sys::env("PATH"))

// Get HOME environment variable
let home = std::sys::env("HOME")
println("Home: ${home}")
Warning: Accessing environment variables may expose sensitive information. Use caution when handling values like passwords or API keys.

Native Functions

Bengal supports calling native functions implemented in other languages (typically Rust) through the FFI.

native fn functionName(params): ReturnType

Declare a native function that is implemented outside of Bengal.

Example 
// Declare a native function
native fn exception_test()

// Call the native function
try {
    exception_test()
} catch (e) {
    println("Caught: " + e)
}
Note: Native functions must be registered with the Bengal runtime before they can be called. See the compiler documentation for details on implementing native functions.
